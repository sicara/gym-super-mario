# gym-super-mario
#### **Gym Super Mario is an environment bundle for OpenAI Gym**
---
Installation
============

```shell
pip install git+https://github.com/sicara/gym-super-mario.git
```

 To load and run the environments, run

```python
import gym
import super_mario
```

To show the emulator window fullscreen add an environment variable `export FULLSCREEN=true`
